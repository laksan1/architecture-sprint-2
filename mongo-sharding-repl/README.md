# Mongo-Sharding-Repl

## Шаги

- все шаги описаны в файле скрипта

```
./scripts/setup_mongo-sharding-repl.sh
```

### Запуск

Запускаем docker compose с контейнерами

```shell
docker compose up -d
```

Запускаем скрипт для инициализации конфига и данных

```shell
./scripts/setup_mongo-sharding-repl.sh
```
